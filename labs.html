Lab 1

Web Usability
10 points I will follow regarding web usability

Aesthetic and minimalist design

1. Make sure dialogue boxes contain only important and relevant information

2. Make use of simple design, with a limited color palette

3. Provide spacing around components allowing all text and information to “breathe”

Consistency and Standards

4. Website features and navigation should be consistent so a user can perform the same actions on different parts of the website (submitting 2 different forms, done the same way)

5. Use identical terminology in prompts, menus, and help screens, as well as consistent color, layout, capitalization, and fonts.

User Control and Freedom

6. Provide a way for a user to navigate back to the previous page

7. Provide a way for a user to navigate back to the main page (emergency exit)

8. Support both redo and undo actions

Font Styles

9. Font should match the theme or intention of the website (don’t use old style font on a sci-fi website)

10. Font colour should not clash with the background it is on (no black text on black background)
